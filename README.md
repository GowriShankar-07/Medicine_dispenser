# 💊 Medicine Dispenser App

A web application for digitizing handwritten prescriptions with OCR and QR code generation.

## Features
- 🔐 Aadhar-based authentication with OTP
- 📸 Camera/image upload for prescriptions
- 🔍 OCR text extraction using EasyOCR
- ✏️ Edit and manage medicines
- 📱 QR code generation for dispensing
- 📋 Prescription history

## Tech Stack
- Streamlit
- EasyOCR
- Python
- QR Code generation

## Run Locally
```bash
pip install -r requirements.txt
streamlit run medicine_app.py
```
